# OCEL 2.0 Event Log Analysis
This repository contains **Object-Centric Process Mining** (OCPM) projects based upon simulated and real-world **Object-Centric Event Logs** (OCEL) structured according to the OCEL 2.0 standard. These event logs are available at https://www.ocel-standard.org/event-logs/overview/. The page's root website, www.ocel-standard.org, maintains the OCEL standard.
